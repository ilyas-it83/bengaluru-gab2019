    GLOBAL AZURE BOOTCAMP
    CHENNAI

    Welcome to the Global Azure Bootcamp event! On Saturday April 22, 2017 we will beat some records.

    In April 2013 the first Global Azure Bootcamp was organized in more than 134 locations around the globe. In 2016, when we organized the third Azure Bootcamp in Chennai, we hosted nearly 200 IT professionals together with the country's top experts on Microsoft cloud. The 2015 edition only increased these record numbers by holding the simultaneous event in 183 locations in 29 countries around the world. In Chennai we were almost 300. Last year we managed to increase that figure and for more than a month of the event, we had to hang the "Sold" poster with more than 450 records. The Azure Bootcamp has been consolidated as the biggest event of the Azure community in Spain and worldwide.

    This year we want to repeat in Chennai one of the venues that will be offered in one day sessions to deepen the use of the Microsoft Azure platform, as well as talks with experts. In addition to this great learning opportunity, attendees will be able to participate in several hands-on "global" labs, forming part of a global computing farm.

    One of the main features of Global Azure Bootcamp has been the realization of a hands-on lab on a global scale. Previous editions have implemented 3D render and animation labs as well as processing genetic datasets to help understand and cure diabetes or breast cancer. In the last edition it was not possible to find an algorithm that met the requirements for this type of massive processing, but ... This year we are working hard so that this does not happen again, and soon we will announce the scientific lab! And believe it, it's going to be VERY VERY interesting

    For the edition of this Bootcamp is going to continue with the same idea of ​​the previous year with two hands-on labs:

    • GAB Science Lab - A research lab focused on data processing. TBA
    • GAB Racing Lab - A gaming lab introducing a 3D driving game with Azure's back-end services. This year We will have some news with a more developed lab than the previous year.
    Our intention is that attendees can use this event to learn how to deploy applications in Azure and to know how distributed computing works, At the same time that we help to solve some problems in the world passing an unforgettable day.
