About The Event

Microsoft Reactor Launch Bangalore 2019 @ Bengaluru is a full-day, event on Microsoft Cloud for Startups. The event is organized by Bengaluru Azure User Group & Bangalore Dotnet Usergroup on 7th September, 2019. It is free and open for everybody to join. This is a one day event to help the developer eco-system from the start-up community and mid size enterprises get up to speed on developing applications on Microsoft Azure.

This conference will feature numerous sessions and workshops delivered by Microsoft MVPs and other expert speakers from the local community, who will chase down the latest developments in Microsoft Azure. The talks and workshops will cover a diverse range of topics including Web,Mobile, DevOps and Microservice deployment using Kubernetes.
